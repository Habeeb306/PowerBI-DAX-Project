# 📊 Power Query Data Cleaning Project

## 📌 Project Overview

This project demonstrates the use of **Microsoft Excel Power Query** for cleaning and transforming a messy retail sales dataset. The dataset was intentionally designed with multiple data quality issues to simulate real-world data preparation challenges.

The goal of this project is to transform raw, inconsistent data into a **structured, clean, and analysis-ready dataset** using Power Query tools.

---

## 📂 Dataset Description

The dataset represents retail sales transactions from different branches across Nigeria. It includes:

- Customer details  
- Product information  
- Payment methods  
- Sales amounts  
- Transaction dates  

### ⚠️ Data Quality Issues Present

The raw dataset contained several issues such as:

- Duplicate records  
- Missing values  
- Incorrect spellings  
- Inconsistent capitalization  
- Mixed data types  
- Irregular date formats  
- Currency symbols in numeric columns  
- Extra spaces in text fields  

---

## 🛠️ Power Query Tasks Performed

### 🧹 Data Cleaning

- Removed duplicate records  
- Replaced incorrect and inconsistent values  
- Trimmed extra spaces from text columns  
- Standardized capitalization formats  
- Handled missing values  

---

### 🔄 Data Transformation

- Split combined columns using delimiters  
- Merged columns  
- Changed data types  
- Created custom columns  
- Added conditional columns  
- Added index columns  
- Used "Column From Examples" feature  

---

### 🔗 Query Operations

- Appended multiple sales tables  
- Merged dimension tables with fact table  

---

## 🗂️ Data Model Structure

### 📌 Fact Table

**Sales_Transactions**

- Contains transactional sales records  
- Serves as the central table for analysis  

---

### 📌 Dimension Tables

**Product_Dimension**  
**Customer_Dimension**

- Enrich the fact table with additional descriptive attributes  
- Used for relational modeling  

---

## 📈 Key Skills Demonstrated

- Data cleaning and transformation using Power Query  
- Handling missing and inconsistent data  
- Data modeling (fact and dimension tables)  
- Merging and appending queries  
- Building analysis-ready datasets  

---

## 🎯 Conclusion

This project demonstrates how **Power Query in Excel** can be used to clean, transform, and prepare messy datasets for analysis. It highlights the importance of proper data preparation in building reliable business intelligence solutions.
